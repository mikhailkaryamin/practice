# Система управления больницей

### Установка
Установить front и back

```
npm install
```

### Запуск сервера

Перед запуском нужно заполнить конфиг .env, пример можно .env.example

```
npm start
```

### Запуск фронта

```
npm start
```

## Библиотеки
- **MERN**
   - **MongoDB**,
   - **Express.js**,
   - **React.js**,
   - **Node.js**
